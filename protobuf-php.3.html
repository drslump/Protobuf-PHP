NAME

protobuf-php - The framework

SYNOPSIS

<?php
require_once 'DrSlump\Protobuf.php';

use DrSlump\Protobuf;

$data = file_get_contents('data.pb');
$person = new Tutorial\Person($data);

DESCRIPTION

Protobuf for PHP is a library to generate, parse and serialize data structures compatible with Google's Protocol Buffers.
